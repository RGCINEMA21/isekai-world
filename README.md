# 🌍 ISEKAI WORLD

RPG Pixel Farming Game - Built with Phaser 3

## 🚀 How to Run

1. Open `index.html` in a web browser
2. Or use a local server (recommended):
   ```bash
   npx serve .
   ```
3. Navigate to `http://localhost:3000/ISEKAI-WORLD/`
